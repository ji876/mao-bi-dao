# mao-bi-dao

> 让 AI 模仿微信公众号"猫笔刀"的写作风格和思维方式

## 这是什么

这是一个 [opencode](https://github.com/anomalyco/opencode) / AI agent skill，定义了财经自媒体博主"猫笔刀"的完整人格档案，包括：

- **写作风格引擎**：标题、开篇、结构、结尾、语言特征的全套规范
- **思维方式**：投资哲学、社会观点、对热点事件的态度
- **评论回复模式**：与读者互动时的口吻和策略
- **专属词汇表**：老登、光模块、碳基/硅基、下水沟、尸体等标志性用语
- **参考语料**：从原文精选的6段落 + 8条评论回复，用于校准语感

## 猫笔刀是谁

微信公众号"猫笔刀"博主，笔名 moomoocat，财经领域头部自媒体。风格特点：

- 接地气的大白话，把复杂财经概念翻译成人话
- 善用比喻拟人（"下水沟里的消费股钻出来喘口气"）
- 反鸡汤，实话实说，不愿画饼
- 带点痞气的幽默反讽
- 独立思考，不盲从巴菲特，不追热点

## 安装

将 `mao-bi-dao/` 目录复制到你的 opencode skills 目录下：

```bash
# Linux / macOS
cp -r mao-bi-dao/ ~/.opencode/skills/

# Windows
xcopy /E /I mao-bi-dao %USERPROFILE%\.opencode\skills\mao-bi-dao
```

## 文件结构

```
mao-bi-dao/
├── SKILL.md              # 主 skill 文件：人格定义、风格规范、思维模式
└── reference-articles.md  # 精选参考语料：原文段落 + 评论回复
```

## 使用方式

安装后，在 opencode 对话中直接让 AI 使用猫笔刀的风格写作或回复即可。例如：

- "用猫笔刀的风格写一篇关于今天A股行情的公众号文章"
- "以猫笔刀的口吻回复这条读者评论"
- "帮我分析一下最近的存储芯片行情，用猫笔刀的方式"

## 效果示例

**标题风格**：4-8字，情绪化口语化 — "血赔10亿"、"又又又爆雷了"、"财富转移太狠了"

**正文风格**：

> 每逢人气板块退潮，下水沟里的消费股就会钻出来喘口气，今天白酒逆势上涨2.1%，那些卧倒的"尸体"们又悄悄的蠕动了下身子，调整了一个自己更舒服的姿势继续装死。

**评论回复风格**：

> 读者：美股这一个月有多猛……当然了 你可以买A股
> 猫笔刀：买a股有投胎功德圈，加仓1块钱+1功德，买港股还有优惠，加1.5功德，累计100万功德下辈子优先插队投好胎

## 兼容性

- [opencode](https://github.com/anomalyco/opencode) — 原生支持
- Claude Code — 放置到 `~/.claude/skills/mao-bi-dao/`
- 其他 AI agent 框架 — 将 SKILL.md 内容作为系统提示注入即可
